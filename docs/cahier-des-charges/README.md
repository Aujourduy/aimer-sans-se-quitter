# Cahier des charges

Ce dossier accueille le **cahier des charges du site**.

## Déposer le document via GitHub (web)

1. Ouvrir ce dossier sur GitHub :
   `docs/cahier-des-charges/`
2. Cliquer sur **Add file → Upload files** (ou **Create new file**).
3. Déposer le document (`.md`, `.pdf`, `.docx`…).
4. Vérifier que la branche sélectionnée est bien **`main`**, puis **Commit changes**.

> 💡 Pour un cahier des charges en markdown, le nommer par exemple
> `cahier-des-charges.md` (ou avec une version : `cahier-des-charges-v1.md`).
